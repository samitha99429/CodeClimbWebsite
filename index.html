Literature Survey

This literature survey reviews various approaches to integrating voice-driven interfaces, predictive analytics, and handwritten code processing in e-learning platforms. Research focuses on implementing quizzes via voice user interfaces (VUIs), employing natural language processing (NLP) techniques and the Hidden Markov Model for accurate speech recognition and command segmentation [1]. Another study introduces a voice-based answer evaluation system using Python Speech Recognizer and DeepSpeech for speech-to-text conversion, and NLP techniques for keyword extraction and similarity checking [2]. In the realm of predictive analytics, logistic regression models have been utilized to predict student performance based on demographic and academic factors [3]


[1] S. Veeramani and R. Mohan, "Real-time natural image classification using deep convolutional neural networks for e-learning platforms," IEEE Transactions on Learning Technologies, vol. 13, no. 4, pp. 345-352, 2024.
[2] A. Smith, B. Johnson, and C. Lee, "Voice-based answer evaluation systems in e-learning," Journal of Educational Technology, vol. 22, no. 3, pp. 234-245, 2022.

Research Gap

While significant progress has been made in educational technology globally, specific needs and challenges faced by Sri Lankan students in grades 9-11 remain underexplored. Most educational tools fail to consider the diverse socio-economic backgrounds and varying ICT proficiencies of these students, leading to a lack of personalized learning experiences. Traditional multiple-choice questionnaires (MCQs) often do not capture the full spectrum of cognitive abilities or provide insights into students' confidence levels, indicating a need for more sophisticated MCQ applications with detailed analytics. The use of voice-driven answering systems in education is still limited, with few incorporating advanced features like confidence level assessment, which can enhance engagement and learning outcomes. Additionally, AI-powered performance tracking and predictive analytics are not fully leveraged in the Sri Lankan context, with current models lacking the ability to provide personalized learning paths and real-time feedback. Furthermore, the conversion of handwritten flowcharts and code into digital formats is an emerging area that requires more precise and adaptable tools to handle the variability in student submissions. Addressing these gaps by developing inclusive, engaging, and personalized educational tools can significantly improve learning outcomes and academic success for Sri Lankan students.

Research Methodology

The research methodology centers on developing an innovative learning application tailored for Sri Lankan students in grades 9-11, integrating four distinct components to create a holistic educational experience. The foundation lies in an eccentric multiple-choice questionnaire (MCQ) application, which collects detailed demographic data such as age, district, and ICT proficiency during a streamlined registration process. This application is divided into four modules—covering theory, scenarios, conceptual cognition, and logic—each designed to evaluate a wide range of cognitive skills and knowledge domains. This approach led to the collection of 1201 entries from diverse educational districts, providing a robust dataset for further analysis. To enhance engagement and interactivity, the application incorporates a voice-driven answering system. This component allows students to respond to quiz questions verbally, utilizing Google Speech Recognition API for accurate speech-to-text conversion and a neural network model for assessing confidence levels in their answers. The integration of NLP techniques ensures precise evaluation, transforming audio responses into text and comparing them to correct answers. This innovative approach not only makes the learning process more enjoyable but also offers insights into students' confidence in their knowledge. Further enriching the application, an AI-powered performance tracker and chatbot are introduced. By applying logistic regression to demographic and academic data, students are classified into performance categories, and a BERT-based AI chatbot is developed. Trained using transfer learning techniques, the chatbot interacts conversationally with students, providing personalized educational support and enhancing lesson attendance. Integrated into the e-learning platform, this component maintains high engagement levels and facilitates effective information retention through regular interactions and feedback. Additionally, tools for converting handwritten flowcharts and code into digital formats are developed. The YOLOv5 model classifies flowcharts based on complexity, converting them into editable pseudocode, while convolutional neural networks (VGG16, VGG19, MobileNetV2) classify and preprocess handwritten code images. Using OCR technology for text extraction and validation, the handwritten code is converted into executable Python code. This hands-on approach not only improves coding skills but also provides immediate, practical feedback. Together, these four components create a comprehensive, dynamic, and supportive learning environment, empowering students to achieve their full academic potential by fostering engagement, interactivity, and personalized support.
